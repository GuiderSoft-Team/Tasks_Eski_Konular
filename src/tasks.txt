TASKS
Task 1: Klavyeden girilen taban (int) ve üs (int) değerlerinin işlem sonucunu
(üs alma işlemi) ekrana yazdıran Java programını yazınız. (Örn; 2^3=8 gibi.)

Task 2: Klavyeden girilen iki değer arasındaki tüm sayılardan sadece üçe tam
bölünenleri tespit edip ekrana yazdıran Java programını yazınız.

Task 3: Verilen iki değer arasındaki tam sayıların ikiye tam bölünenlerini

Task 5: Verilen sayının aralık dışında olup olmadığını sorgulayan metot yazınız.
Method: boolean isNumberOutOfRangeFiftyAndHundred(int number)

Task 6: Verilen bir mantıksal dizide kaç tane doğru bulunduğunu döndüren

Method: int countTrueValues(boolean[] values)

Task 7: Verilen bir karakter dizisini metin olarak döndüren metot yazınız.
Method: String makeText(char[] values)

Method: char[] convertStringToChars(String text)

Method: int signumFunc(int value)
